Documentation

Selamat datang di CineData API. Platform ini menyediakan akses data film publik melalui REST API. Dokumentasi ini membantu kamu untuk mengintegrasikan data film ke dalam aplikasi kamu dengan cepat.

Base URL: http://localhost:3000/api/v1

Authentication

API ini menggunakan API Key untuk mengautentikasi setiap request. Kamu bisa mendapatkan key kamu di Dashboard Developer.

Kirimkan key kamu melalui header x-api-key:

x-api-key: YOUR_API_KEY_HERE
GET

/movies

Mengambil daftar semua film yang tersedia di database CineData.

Example Request (JavaScript)


fetch('http://localhost:3000/api/v1/movies', {
  headers: {
    'x-api-key': 'wf_da8e16...'
  }
})
.then(res => res.json())
.then(data => console.log(data));

Example Success Response


{
  "status": "success",
  "results": 1,
  "data": [
    {
      "id": 1,
      "title": "Inception",
      "release_year": 2010,
      "genres": "Action,Sci-Fi",
      "rating": "8.8",
      "director": "Christopher Nolan"
    }
  ]
}

Error Codes

Code Description
401 API Key missing (Header x-api-key tidak ditemukan).
403 Invalid API Key (Key salah atau tidak aktif).
429 Too Many Requests (Kamu melewati batas rate limit).
404 Endpoint not found.
