Getting Started

Selamat datang di CineData API. Kami menyediakan akses ke metadata film dark thriller dan drama Korea paling komprehensif. Gunakan instruksi di bawah untuk memulai integrasi aplikasi Anda.

Base Cloud URL

http://localhost:3000/api/v1

Authentication

Seluruh permintaan ke CineData API wajib menyertakan kunci akses unik (API Key). Anda dapat mengelola kunci Anda melalui Developer Dashboard.

Required Header

x-api-key: wf_da8e16a54349...
Get

/movies

Mengambil daftar seluruh katalog film, series, dan drakor dalam database publik kami.

Request Example

fetch('/api/v1/movies', {
  headers: { 'x-api-key': 'YOUR_KEY' }
})
Get

/movies/:id

Mengambil metadata detail spesifik (Termasuk Cast & Synopsis) berdasarkan ID film.

Success Response

{
  "status": "success",
  "data": {
    "id": 1,
    "title": "A Shop for Killers",
    "rating": "8.1"
  }
}

Error Codes

CodeMeaningFix
